# Rps
Rock-paper-scissors game made with FXML-based javaFX.

![](http://i.imgur.com/HDnL2yt.png)

## Running Rps
Navigate to the project directory and run the program by

On Mac & Linux
```
./gradlew run
```
On Windows
```
gradlew run
```
